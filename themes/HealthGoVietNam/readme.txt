HealthGoVietNam - WordPress Theme

A modern and clean WordPress theme with responsive design.

== Installation ==

1. Upload the theme folder to the /wp-content/themes/ directory
2. Activate the theme through the 'Appearance' menu in WordPress
3. Go to Appearance > Customize to configure your theme settings

== Features ==

* Responsive Design
* Custom Logo Support
* Custom Header
* Custom Background
* Navigation Menus (Primary and Footer)
* Widget Areas (Sidebar and 3 Footer areas)
* Featured Images
* Post Thumbnails
* Threaded Comments
* Translation Ready
* HTML5 Markup
* SEO Friendly

== Customization ==

This theme supports the WordPress Customizer for easy customization:
- Site Identity (Logo, Site Title, Tagline)
- Colors
- Header Image
- Background Image
- Menus
- Widgets

== Menus ==

This theme has 2 menu locations:
1. Primary Menu - Main navigation in the header
2. Footer Menu - Secondary navigation in the footer

== Widget Areas ==

This theme has 4 widget areas:
1. Sidebar - Main sidebar widget area
2. Footer 1 - First footer widget area
3. Footer 2 - Second footer widget area
4. Footer 3 - Third footer widget area
